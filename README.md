# Smart Port Scanner (Honeypot-Aware)

A unique port scanning tool that detects open ports, service banners, and honeypot behavior.

## A Python-based port scanner that performs:
- TCP port scanning
- HTTPS certificate fingerprinting
- Banner grabbing
- Basic honeypot detection
- JSON logging of scan results

## Installation
```bash
# Clone the repository
git clone https://github.com/l0h1nth/Honeypot-Aware-Port-Scanner.git

# Enter the directory
cd Honeypot-Aware-Port-Scanner

# Run:
python3 scanner.py
```
